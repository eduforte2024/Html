Bem-vindo(a) ao meu repositório de práticas de Front-End! Aqui é o meu "laboratório" pessoal, onde guardo todos os meus códigos, testes e experimentos enquanto aprendo HTML, CSS e tecnologias Web.

---

## 🚀 Objetivo do Repositório

O foco principal deste espaço é **praticar e evoluir**. Uso este repositório para:
* 🧪 Criar páginas de teste e rascunhos rápidos.
* 📚 Fixar conceitos de tags estruturais, formulários e semântica do HTML5.
* 🎨 Brincar com estilizações e layouts simples.
* 🔄 Entender o fluxo de versionamento usando o Git e GitHub.

---

## 📂 O que você vai encontrar por aqui?

* **Páginas Estruturais:** Exercícios de fixação de cabeçalhos, parágrafos, listas e tabelas.
* **Formulários:** Testes de campos de entrada (`inputs`), botões e validações nativas.
* **Projetos Pessoais:** Pequenas páginas completas criadas do zero para aplicar o conhecimento na prática.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Para a estruturação semântica das páginas.
* **VS Code:** Meu editor de código principal para o desenvolvimento.
* **Git & GitHub:** Para salvar o progresso e manter o histórico de alterações.
